## Development

To start developing locally:

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm run dev
   ```
   This will:
   - Watch and rebuild Tailwind CSS
   - Serve the site at [http://localhost:5173](http://localhost:5173)
